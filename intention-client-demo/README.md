# Intention Client (demo)
This tiny SDK demonstrates computing contentHash, signing the intention (HMAC demo) and sending
the intention+payload to the Object proxy.

Quickstart:
1. In this directory run: `chmod +x run_iptp.sh && ./run_iptp.sh`
2. Then run `npm run send` to execute the demo `test-send.js` script.
